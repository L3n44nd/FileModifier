Обработчик файлов 

Сборка:
- Открыть CMakeLists.txt в Qt Creator
- Выбрать комплект MinGW
- Нажать "Собрать"

Функциональность: 
- пользовательский выбор входной и выходной директорий
- режимы: единичный или по таймеру
- модификация файлов XOR с 8-байтным ключом
- чтение больших файлов буферами (10 МБ)
- отдельный поток для обработчика файлов
- шкала прогресса
- возможность мгновенной остановки без повреждения файлов
- валидация полей 

Cтруктура: 
- fileModifier - главное окно 
- fileProcessor - обработчик
